# 🌿 Restorative Intelligence: Eco-Impact Tracker

**Restorative Intelligence** is a high-fidelity, AI-powered carbon footprint tracker designed for institutional ESG analysts and eco-conscious individuals. It transforms raw transaction data into actionable sustainability insights using advanced classification and premium bento-style visualizations.

![Eco Impact Dashboard Mockup](https://images.unsplash.com/photo-1473341304170-971dccb5ac1e?auto=format&fit=crop&q=80&w=2000)

## 🚀 Features

### 🧠 AI-Powered Transaction Classification
- **Smart Categorization**: Automatically classifies bank statements/CSV uploads into categories like Fuel, Electricity, Flights, and Shopping.
- **Emission Analysis**: Calculates real-time CO2 impact based on India-specific emission factors (kg/INR).
- **Processing Meta**: Provides confidence scores and "AI-assisted" flags for every processed batch.

### 📊 Institutional Dashboard
- **Bento Grid Layout**: High-fidelity stats cards showing Carbon Footprint, Green Score, CO2 Saved, and Rewards.
- **Trend Visualization**: Modern bar charts tracking monthly emission trends.
- **Environmental Ledger**: A detailed table view of all classified transactions with per-item impact analysis.

### 💡 AI Insights & Suggestions
- **Targeted Advisory**: Personalized recommendations to reduce carbon footprint (e.g., EV Fleet transitions, HVAC optimization).
- **Impact Metrics**: Clear "CO2 Reduction %" and "Potential Savings" for every suggestion.
- **Merchant Partnerships**: Integrated promo codes for sustainable partners (Solar, Recycling, etc.).

### 🏆 Rewards & Gamification
- **Paytm Gold Rewards**: Earn real value for eco-friendly choices. Claim rewards as you hit CO2 mitigation milestones.
- **Institutional Leaderboard**: Compare sustainability performance across different entities or teams.
- **Verified Achievements**: A gallery of earned badges (Forest Guardian, Solar Sage, etc.) based on real-world impact.

## 🛠 Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Design System**: [Google Stitch](https://stitch.google.com/) (Restorative Intelligence Theme)
- **Charts**: [Recharts](https://recharts.org/)
- **Icons**: [Google Material Symbols](https://fonts.google.com/icons)
- **AI Simulation**: Integrated mock-AI backend for transaction processing and classification.

## 📦 Getting Started

### 1. Installation
```bash
npm install
```

### 2. Environment Setup
Create a `.env` file for any API keys (e.g., Gemini API for real LLM classification in future steps).

### 3. Run Development Server
```bash
npm run dev
```

### 4. Open Application
Visit [http://localhost:3000](http://localhost:3000) to see your restorative dashboard.

## 🎨 Design System

This project uses the **Restorative Intelligence** theme, featuring:
- **Glassmorphism**: Translucent, blurred cards for a premium feel.
- **Typography**: Inter (Display) & Space Grotesk (Mono Data).
- **Color Palette**: Deep emerald greens, teal accents, and warm gold highlights.
